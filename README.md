# Neeraj Chaurasiya — Portfolio

Personal portfolio of **Neeraj Chaurasiya**, Senior Software Engineer specializing in 5G Telecom, Zero Trust Security, Embedded Linux, and AI Agents.

🔗 **Live:** [neerajchaurasiya.dev](https://neerajchaurasiya.dev)

## Tech Stack

- **Framework:** React + TypeScript + Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Design:** Nordic Minimalism

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

Deployed automatically via **Cloudflare Pages** on every push to `main`.

| Setting | Value |
|---|---|
| Build command | `npm run build` |
| Output directory | `dist` |
| Node version | 18 |
